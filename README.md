# Chase Runner 🏃

> A fast-paced, dual-character chase runner game built with MonoGame and C#.

![Game Screenshot](https://via.placeholder.com/800x400?text=Chase+Runner+Gameplay) *(Görseller ve oyun içi GIF'ler daha sonra eklenecek)*

## 🎮 About The Game
Chase Runner is an interactive 2D runner game focusing on dual-character mechanics and fast-paced gameplay. Originally developed as a game programming project, it features custom physics, collision detection, and tight controls.

## 🛠️ Tech Stack
- **C#**
- **MonoGame Framework**

## 🚀 Getting Started
1. Clone this repository: `git clone https://github.com/CadaKerem/chase-runner.git`
2. Open the `.sln` solution file in **Visual Studio**.
3. Restore NuGet packages and run the project!
